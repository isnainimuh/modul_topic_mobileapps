Instalasi Flutter dan Android Studio

Sebelum memulai membuat project Flutter, lakukan instalasi Flutter SDK dan Android Studio lebih dulu. Panduan instalasi tidak akan dibahas di bab ini, silakan mencari dokumentasinya di situs resmi atau sumber lain yang terpercaya.

Create Flutter Project

Pada bab ini, pastikan sudah menginstal Flutter SDK dan Android Studio dan sudah siap digunakan. Kita akan membuat project frontend ecommerce dengan Flutter. Masuk dalam terminal laragon 'C:\laragon\www' dan buat folder project didalamnya.

Step 1. Buat folder project di dalam laragon/www


flutter create ecommerce_mobile
# pindah ke direktori project
cd ecommerce_mobile
# Test run di Chrome
flutter run -d chrome

Jika Berhasil, stop dengan tekan q di terminal, lalu buka di VS Code dengan mengetikkan 'code .' di terminal. 


code .

Step 2. Update pubspec.yaml

Ganti isi pubspec.yaml dengan kode berikut.

Jalankan di terminal:


flutter pub get

Step 3. Buat Folder Structure

Di VS Code, buat folder baru di lib/:

  • models
  • services
  • screens
  • widgets
