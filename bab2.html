Step 1. Membuat Seeder

Langkah berikutnya kita buat Seeder untuk data dummy produk. Kita akan buat seeder untuk user, categories, products, dan product_images.

Buat Seeder untuk User


php artisan make:seeder UserSeeder

Buat Seeder untuk Categories


php artisan make:seeder CategorySeeder

Buat Seeder untuk Products


php artisan make:seeder ProductSeeder

Buat Seeder untuk Product Images


php artisan make:seeder ProductImageSeeder

Update Database Seeder Utama

Lokasi file: database/seeders/DatabaseSeeder.php

Jalankan Seeder


php artisan migrate:refresh --seed

Step 2. Membuat Controller

Tahap berikutnya kita akan membuat controller untuk handle logic aplikasi. Kita mulai dengan beberapa logic yang essential yaitu ProductContoller untuk handle produk, CartController untuk handle keranjang, OrderController untuk handle order dan PaymentController untuk handle pembayaran.

Product Controller


php artisan make:controller ProductController

Cart Controller


php artisan make:controller CartController

Order Controller


php artisan make:controller OrderController

Payment Controller


php artisan make:controller PaymentController

Edit file routes

Lokasi file: routes/api.php
